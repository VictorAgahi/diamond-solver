# Diamond Solver

Petit solveur Python pour le probleme de blueprints avec l'extension diamond.

## Contenu

- `submission.py` : solveur principal
- `seed.txt` : exemple d'entree a 2 blueprints
- `Presentation.MD` : explication complete de l'algo avec schemas Mermaid

## Setup

1. Installer Python 3.10+.
2. Se placer a la racine du projet.
3. Lancer le script avec l'entree voulue.

```bash
python3 submission.py seed.txt
```

## Fonctionnement

Le script lit chaque blueprint, calcule le maximum de diamonds en 24 minutes pour la partie 1, puis affiche aussi la partie 2 avec les 3 premiers blueprints et 32 minutes.
